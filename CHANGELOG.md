## [1.0.0] - 2026-07-18

### Added

- Project management and weekly allocation planning
- Stopwatch and countdown tracking from the macOS menu bar
- Pause/resume focus sessions and automatic timer recovery
- Native timer and allocation notifications
- Manual activity correction, splitting, and deletion
- Markdown notes, GitHub work links, reusable tags, search, and filtering
- Configurable weekly capacity and launch-at-login support
- Local versioned Core Data persistence and migration verification
